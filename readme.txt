== Description ==

Allows to delete all comments with single click. you can select all unapproved and pending comments or comments based on category / post.

== Installation ==

Just a standard installation required.

1. Plugins > Add New > Upload > 'bulk-delete-comments.zip' - Install Now

2. Unzip & Upload 'bulk-delete-comments.zip' folder to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress
